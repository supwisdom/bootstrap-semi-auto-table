Events

Event Type Description
pageChange 翻页的时候触发。 
$('#table').on('pageChange', function (event, pageObject) {
  // do something
});
sortChange 排序发生变化的时候触发。传入的 sortObject仅仅包含了排序发生变化的部分，而不是所有。 
$('#table').on('sortChange', function (event, sortObject) {
  // do something
});
